# Metadata Store

**A database-hosted metadata repository and data dictionary**

---

This project defines a [data dictionary](https://en.wikipedia.org/wiki/Data_dictionary) and metadata repository hosted in an [ArangoDB](https://www.arangodb.com/) database and accessible via [Foxx](https://www.arangodb.com/docs/stable/foxx.html) [API](https://en.wikipedia.org/wiki/Application_programming_interface). The repository contains the data and code needed to populate the data dictionary core and ancillary data standards.

*More detailed documentation and usage will be added here.*
