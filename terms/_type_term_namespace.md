# `_type_term_namespace`

**`_title`**

Namespace

**`_definition`**

The term type indicating that the term is a namespace — a grouping container providing the `_nid` value for other terms.

**`_description`**

A term marked `_type_term_namespace` provides the [`_nid`](_nid.md) value for other terms in the dictionary. It defines a shared namespace scope rather than describing a data value or schema. Namespace terms typically carry no [`_data`](_data.md) or [`_rule`](_rule.md) section.
