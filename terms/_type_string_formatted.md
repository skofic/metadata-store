# `_type_string_formatted`

**`_title`**

Formatted text

**`_definition`**

Edge section grouping formatted text string types: HTML, Markdown, LaTeX, and SVG. Not a valid value for any type property.

**`_description`**

`_type_string_formatted` is a non-selectable edge section used in graph display to group [`_type_string_HTML`](_type_string_HTML.md), [`_type_string_Markdown`](_type_string_Markdown.md), [`_type_string_LaTeX`](_type_string_LaTeX.md), and [`_type_string_SVG`](_type_string_SVG.md) under a common collapsible heading. It is not a valid value for [`_scalar_type`](_scalar_type.md), [`_comparable_type`](_comparable_type.md), or [`_dict_key_type`](_dict_key_type.md). Formatted string types are excluded from [`_type_data_comparable`](_type_data_comparable.md).
