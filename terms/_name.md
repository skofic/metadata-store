# `_name`

**`_title`**

Name

**`_definition`**

The human-readable name of the term, used when the term has an official or well-established name that is distinct from its identifier.

**`_description`**

This property holds a *name* for the term — a human-readable label that is more descriptive or recognisable than the [local identifier](_lid.md). It is typically used in two situations:

- When the term has an *official denomination* in its domain (for example, the full name of a country in its official or native language, or the registered name of a standard).
- When the identifier alone would be opaque to a reader unfamiliar with the naming convention.

This property is a plain string with no language tag: it records the name as it naturally appears, without referencing a specific language. Multilingual labels belong in the [documentation section](_info.md)'s [title](_title.md) property instead.

**`_examples`**

粵語 / 粤语 — the native name of the Cantonese language, written in Cantonese script.
