# `_key`

**`_title`**

Document key

**`_definition`**

The unique identifier of the document within its collection.

**`_description`**

The *document key* is the *unique identifier* of a document *within its collection* in [ArangoDB](https://www.arangodb.com), the default database for this data dictionary. In general ArangoDB usage it can be supplied by the user or auto-generated by the database. In this dictionary, the key of a [term](_term.md) document is always set by the loader from the term's [global identifier](_gid.md) — it is never provided directly by the user or auto-generated.

---

**`_code`**

```json
{
  "_aid" : [
    "key"
  ],
  "_gid" : "_key",
  "_lid" : "key",
  "_nid" : ""
}
```

**`_data`**

```json
{
  "_scalar" : {
    "_type" : "_type_key"
  }
}
```
