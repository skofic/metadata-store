# `_enum`

**`_title`**

Enumeration

**`_definition`**

Namespace for enumeration-related descriptor terms. Groups `_enum_types` and other terms associated with enumeration-typed scalar values.

**`_description`**

`_enum` is a namespace term that groups descriptor terms used alongside enumeration scalar types. It has no data or schema of its own.

The primary child term is [`_enum_types`](_enum_types.md), which constrains the controlled vocabulary from which an enumeration value must be drawn. This namespace is used wherever [`_scalar_type`](_scalar_type.md), [`_comparable_type`](_comparable_type.md), or [`_dict_key_type`](_dict_key_type.md) holds a [`_type_enum`](_type_enum.md) value.
