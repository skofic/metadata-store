# `_from`

**`_title`**

Relationship source

**`_definition`**

Reference to the node that represents the origin in a directed graph edge.

**`_description`**

The value is the [document handle](_id.md) of the source node of a directed graph relationship. In [ArangoDB](https://www.arangodb.com), `_from` is a required field on all edge documents.

---

**`_code`**

```json
{
  "_aid" : [
    "from"
  ],
  "_gid" : "_from",
  "_lid" : "from",
  "_nid" : ""
}
```

**`_data`**

```json
{
  "_scalar" : {
    "_type_scalar" : "_type_handle"
  }
}
```
