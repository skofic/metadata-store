# `_type_string_internet`

**`_title`**

Internet strings

**`_definition`**

Edge section grouping internet address string types: URI, Email, Hostname, IPv4, and IPv6. Not a valid value for any type property.

**`_description`**

`_type_string_internet` is a non-selectable edge section grouping internet-format string types under a common heading in graph display. Members are [`_type_string_URI`](_type_string_URI.md), [`_type_string_Email`](_type_string_Email.md), [`_type_string_Hostname`](_type_string_Hostname.md), [`_type_string_IPv4`](_type_string_IPv4.md), and [`_type_string_IPv6`](_type_string_IPv6.md). All members are comparable and valid as dictionary keys.
