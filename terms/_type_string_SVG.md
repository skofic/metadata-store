# `_type_string_SVG`

**`_title`**

SVG

**`_definition`**

The scalar data type for SVG image markup stored as a string. The type carries its own format constraint; `_regexp` is not applicable.

**`_description`**

`_type_string_SVG` declares that the value is an [SVG](https://en.wikipedia.org/wiki/SVG) image encoded as a string. No [`_regexp`](_regexp.md) is applicable. Excluded from [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md).
