# `_type_string_Markdown`

**`_title`**

Markdown

**`_definition`**

The scalar data type for Markdown-formatted text, rendered in frontends using a Markdown parser. The type carries its own format constraint; `_regexp` is not applicable.

**`_description`**

`_type_string_Markdown` declares that the value is a [Markdown](https://en.wikipedia.org/wiki/Markdown) string. No [`_regexp`](_regexp.md) is applicable. Excluded from [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md) for the same reasons as [`_type_string_HTML`](_type_string_HTML.md).
