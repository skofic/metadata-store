# `_type_string_HTML`

**`_title`**

HTML

**`_definition`**

Enumeration element of [`_type_scalar`](_type_scalar.md) for HTML-formatted text. The value is a plain UTF-8 string containing HTML markup, intended to be rendered by a browser or HTML processor before display.

**`_description`**

When [`_type_scalar`](_type_scalar.md) is `_type_string_HTML`, the descriptor value is a UTF-8 string containing valid HTML markup. It is used when richer formatting than Markdown supports is required — for example, embedded tables, custom attributes, or inline styles.

No companion properties are applicable.

		"_type_scalar": "_type_string_HTML"
	}
}
