# `_type_string_IPv4`

**`_title`**

IPv4 Address

**`_definition`**

The scalar data type for IPv4 addresses conforming to the JSON Schema `ipv4` format.

**`_description`**

`_type_string_IPv4` declares that the value is an [IPv4 address](https://en.wikipedia.org/wiki/IPv4) (dotted-decimal notation, e.g. `192.168.0.1`). A member of [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md).
