# `_type`

**`_title`**

Types

**`_definition`**

Namespace grouping all type terms in the data dictionary: scalar data types, term type classifications, and selection rule types.

**`_description`**

`_type` is the root namespace for all type-related terms. It organises three main sub-groups: data types ([`_type_data`](_type_data.md) and its sub-enumerations), term type classifications ([`_type_term`](_type_term.md)), and selection rule types ([`_type_selection`](_type_selection.md)). It has no data or schema of its own.
