# `_type`

**`_title`**

Type

**`_definition`**

The namespace root for the dictionary's built-in typedef terms. Groups the three typedef terms that encode the allowed type sets for scalar, set-element, and dictionary-key contexts.

**`_description`**

`_type` is the namespace under which the three built-in typedef terms are organised: [`_type_scalar`](_type_scalar.md), [`_type_set`](_type_set.md), and [`_type_key`](_type_key.md). These terms carry the complete allowed-type lists used when validating the type key present in [`_scalar`](_scalar.md), [`_set`](_set.md), [`_nested`](_nested.md), and [`_dict_key`](_dict_key.md) objects.

As a namespace root, `_type` itself carries no data section.
