# `_type_data_dict`

**`_title`**

Dictionary data types

**`_definition`**

Namespace grouping dictionary key type terms. Its primary child is `_type_data_dict_key`.

**`_description`**

`_type_data_dict` is a namespace grouping the type terms related to dictionary keys. It has no data or schema of its own. Its sole child is [`_type_data_dict_key`](_type_data_dict_key.md), the enumeration root for all dictionary key types.
