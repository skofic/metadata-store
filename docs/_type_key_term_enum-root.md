# `_type_key_term_enum-root`

**`_title`**

Enumeration root key

**`_definition`**

A term key that must reference an enumeration root — a term carrying `_type_term_enum-root` in its `_term_type`.

**`_description`**

`_type_key_term_enum-root` is used when a property value must be the key of an enumeration root term. Typical use: the [`_enum_types`](_enum_types.md) companion property, which holds a set of vocabulary root keys that constrain an `_type_enum` value.

---

**`_code`**

```json
{
  "_aid" : [
    "enum-root"
  ],
  "_gid" : "_type_key_term_enum-root",
  "_lid" : "enum-root",
  "_nid" : "_type_key_term"
}
```
