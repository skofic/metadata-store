# `_type_string_date`

**`_title`**

Date

**`_definition`**

The scalar data type for full dates in ISO 8601 `YYYY-MM-DD` format (JSON Schema `date` format). Comparable; string range constraints apply.

**`_description`**

`_type_string_date` stores a full calendar date with hyphen separators. Lexicographic ordering is meaningful. String range constraints may be applied via [`_range_valid_string`](_range_valid_string.md) and [`_range_normal_string`](_range_normal_string.md). A member of [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md).
