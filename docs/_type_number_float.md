# `_type_number_float`

**`_title`**

Float

**`_definition`**

The scalar data type for floating-point numbers. Values must be stored and treated as floating-point; integer values are accepted but treated as floats. `_decimals` is permitted.

**`_description`**

`_type_number_float` is a sub-type of [`_type_number`](_type_number.md). It requires the value to be stored and displayed as a floating-point number. Integer values are accepted but treated as floats. [`_decimals`](_decimals.md) is permitted to control display precision. A member of [`_type_data_comparable`](_type_data_comparable.md).
