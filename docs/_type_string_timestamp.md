# `_type_string_timestamp`

**`_title`**

Date and time strings

**`_definition`**

Edge section grouping date and time string types: YMD, date, time, and date-time. Not a valid value for any type property.

**`_description`**

`_type_string_timestamp` is a non-selectable edge section grouping date and time format types under a common heading in graph display. Members are [`_type_string_YMD`](_type_string_YMD.md), [`_type_string_date`](_type_string_date.md), [`_type_string_time`](_type_string_time.md), and [`_type_string_date-time`](_type_string_date-time.md). All members support lexicographic ordering and string range constraints.
