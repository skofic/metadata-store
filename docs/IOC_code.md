## [Information section](_info.md)

**[Title](_title.md):** IOC code

**[Definition](_definition.md):** The three-letter code assigned by the International Olympic Committee to identify a National Olympic Committee in competition.

**[Description](_description.md)**

IOC codes are uppercase three-letter strings. They resemble but are not identical to ISO 3166-1 alpha-3 codes — several countries use different codes in the two systems. Not every sovereign country has an IOC code; conversely, some dependent territories have their own National Olympic Committee and therefore their own code.
