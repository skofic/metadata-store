# `_type_key_term_object`

**`_title`**

Object key

**`_definition`**

A term key that must reference an object definition term — a term carrying a `_rule` section.

**`_description`**

`_type_key_term_object` is used when a property value must be the key of an object definition term. Typical use: the [`_object_types`](_object_types.md) companion property, which holds a set of object definition keys constraining a [`_type_object`](_type_object.md) value.

---

**`_code`**

```json
{
  "_aid" : [
    "object"
  ],
  "_gid" : "_type_key_term_object",
  "_lid" : "object",
  "_nid" : "_type_key_term"
}
```
