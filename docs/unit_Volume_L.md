## [Information section](_info.md)

**[Title](_title.md):** Litre (L)

**[Definition](_definition.md):** Unit of volume equal to 10⁻³ m³ (one cubic decimetre).

**[Description](_description.md)**

The litre is a non-SI unit accepted for use with the SI. It equals exactly one cubic decimetre (1 L = 0.001 m³). The uppercase symbol L is preferred to avoid confusion with the digit 1.
