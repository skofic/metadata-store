# `_type_number_integer`

**`_title`**

Integer

**`_definition`**

The scalar data type for integer numbers. Values must have no decimal part; floating-point numbers are rejected. `_decimals` is not permitted.

**`_description`**

`_type_number_integer` is a sub-type of [`_type_number`](_type_number.md). It restricts values to whole numbers — no decimal part is allowed. [`_decimals`](_decimals.md) is not permitted. A member of [`_type_data_comparable`](_type_data_comparable.md).
