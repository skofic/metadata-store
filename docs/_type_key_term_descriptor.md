# `_type_key_term_descriptor`

**`_title`**

Descriptor key

**`_definition`**

A term key that must reference a descriptor — a term carrying a `_data` section.

**`_description`**

`_type_key_term_descriptor` is used when a property value must be the key of a descriptor term. Typical use: the [`_tuple_type`](_tuple_type.md) property, whose elements are descriptor keys defining the type at each tuple position.

---

**`_code`**

```json
{
  "_aid" : [
    "descriptor"
  ],
  "_gid" : "_type_key_term_descriptor",
  "_lid" : "descriptor",
  "_nid" : "_type_key_term"
}
```
