# `_type_string_Email`

**`_title`**

E-mail address

**`_definition`**

The scalar data type for email addresses conforming to the JSON Schema `email` format. The type carries its own format constraint; `_regexp` is not applicable.

**`_description`**

`_type_string_Email` declares that the value is an email address. The type itself defines the format; [`_regexp`](_regexp.md) is not applicable. A member of [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md).
