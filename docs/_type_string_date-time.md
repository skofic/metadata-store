# `_type_string_date-time`

**`_title`**

Date and time

**`_definition`**

The scalar data type for combined date-time values in ISO 8601 format (`YYYY-MM-DDTHH:MM:SS` and variants, JSON Schema `date-time` format). Comparable; string range constraints apply.

**`_description`**

`_type_string_date-time` stores a combined date and time value with the [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) `T` separator. Lexicographic ordering is meaningful. A member of [`_type_data_comparable`](_type_data_comparable.md) and [`_type_data_dict_key`](_type_data_dict_key.md).
