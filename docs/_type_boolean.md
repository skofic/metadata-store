# `_type_boolean`

**`_title`**

Boolean

**`_definition`**

The scalar data type for native boolean values. The stored value is `true` or `false`; no companion properties (range, unit, regexp) are applicable.

**`_description`**

`_type_boolean` represents a native boolean. No companion properties — range, unit, or regexp — apply to boolean values. It is a member of [`_type_data`](_type_data.md) and [`_type_data_comparable`](_type_data_comparable.md), but not of [`_type_data_dict_key`](_type_data_dict_key.md) (booleans cannot serve as object property names).
